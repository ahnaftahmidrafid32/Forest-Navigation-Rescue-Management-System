# 🌲 Forest Navigation & Rescue Management System

## 📌 Overview

The **Forest Navigation & Rescue Management System** is a Java-based application designed to assist in efficient navigation, hazard monitoring, and rescue operations within forest environments. The system integrates multiple modules such as survivor tracking, rescue team coordination, weather updates, and intelligent pathfinding to ensure safety and quick response during emergencies.

This project uses a **file-based storage system** and features a **user-friendly GUI built with Java Swing**, making it simple, lightweight, and suitable for academic and practical use.

---

## 🚀 Features

### 🧭 Navigation & Pathfinding

* Calculates the **safest and fastest path** between two points in the forest
* Considers hazards, terrain difficulty, and weather conditions
* Helps users avoid dangerous zones

### ⚠️ Alerts & Hazard Management

* Real-time alert system for:

  * Wildfires
  * Flooded areas
  * Dangerous wildlife zones
* Hazard data is stored and updated using file-based storage


### 🌦️ Weather Information System

* Displays current and updated weather conditions
* Influences navigation and rescue decisions
* Allows manual weather condition updates


### 🧍 Survivor Management

* Tracks missing or stranded individuals
* Stores survivor details such as:

  * Location
  * Status
  * Priority level



### 🚑 Rescue Management System

* Organizes and assigns rescue missions
* Maintains rescue logs and mission status
* Coordinates between survivors and rescue teams


### 👨‍🚒 Rescue Team Management

* Manages multiple rescue teams
* Tracks team availability, location, and assignments
* Helps in efficient deployment during emergencies


### 💾 File-Based Storage

* All data (users, hazards, survivors, teams, etc.) is stored in files
* No database dependency
* Easy to understand and implement for academic purposes


### 🖥️ Graphical User Interface (GUI)

* Built using **Java Swing**
* Clean, simple, and interactive interface
* Easy navigation between modules
